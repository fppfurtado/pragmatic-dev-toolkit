# audits/runs

Registros de execuções dos prompts de auditoria reutilizáveis em `docs/audits/*.md` (`prose-tokens.md`, `architecture-logic.md`).

Cada arquivo aqui corresponde a uma execução nomeada (data + foco), preservada para arqueologia editorial — o que foi auditado, quais findings emergiram, quais viraram decisão.

Este README documenta o propósito do diretório; o conteúdo é populado conforme auditorias rodam.
